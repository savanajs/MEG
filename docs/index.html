Style Guide

MEG is an open source HTML, CSS and JS development kit based on Flexbox.
Developed with variables, mixins, Sass, responsive grid system
for building layouts for ecommerce.

Why use?

  • Compatibility of elements between browsers (Resolves cross browser component issues)
  • Grid system with 12 colums
  • Over 200 helper classes
  • Accessibility of the elements with WAI-ARIA
  • Light and small framework. meg.min.css (99kb) and meg.app.min.js (11kb)
  • Standardization of elements
  • Easy customization
  • Uses SASS + Webpack
  • Possibility of placing lazyload in images
  • You over write less CSS and create more layouts.

Instalation

Just add the following link element into your page's head, before your project's stylesheets and add the following script element into your page's body.


    <!DOCTYPE html>
    <html lang="pt-BR">
    <head>  
        <meta charset="UTF-8"/>
        <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
        <meta http-equiv="X-UA-Compatible" content="ie=edge"/>
        <title>MEG CSS Framework</title>
        <link href="./docs/meg.min.css" rel="stylesheet" />
    </head>
    <body>
        <script src="https://code.jquery.com/jquery-2.2.4.min.js" integrity="sha256-BbhdlvQf/xTY9gja0Dq3HiwQF8LaCRTXxZKRutelT44=" crossorigin="anonymous"></script>

            <script src="//cdnjs.cloudflare.com/ajax/libs/es5-shim/4.2.0/es5-shim.min.js"></script>

Structure of the project

Mass structure of project files using a BEM methodology.

  • base - Basic styling directly in html tags.
  • _all.scss - Importer of base files.
  • generic.scss - Here are the structure rules of your html tags.
  • resets.scss - Html tags element resets.
  • components - Modules that make up something bigger, that which goes into the composition of the element.
  • _all.scss - Importer of components files.
  • Classes with suffix meg-c-.
  • elements - Basic style of most used HTML elements
  • _all.scss - Importer of elements files.
  • Classes with suffix meg-el-.
  • grid - Structure of your HTML
  • _all.scss - Importer of grid files.
  • columns.scss - Rules of grids.
  • Classes with suffix meg-lt-.
  • layout - Layout rules.
  • _all.scss - Importer of layout files.
  • Classes with suffix meg-lt-.
  • settings - These are comme basic settings of your architecture.
  • _all.scss - Importer of settings files.
  • colors-dark.scss - In general, they are the global variables of the colors of the elements.
  • colors.scss - In general, they are the global variables of the colors of the elements.
  • imports.scss - Importers of internal and external CSS.
  • variables.scss - In general, they are the global variables of the elements.
  • themes - Custom theme.
  • _all.scss - Importer of themes files.
  • default.scss - Here are the colors and styles of your theme.
  • utilities - Helpers and Modifiers
  • _all.scss - Importer of utilities files.
  • state.scss - State of a Modifier (Classes with suffix meg-s-).
  • tools.scss - Here are the mixins.
  • utils.scss - Assistant classes (Classes with suffix meg-u-).
